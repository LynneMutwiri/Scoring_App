# SACCO Credit Scoring Dashboard

This project implements a machine learning-based credit scoring system designed for SACCOs.

## Features
- Borrower-level risk prediction
- Portfolio insights visualization
- Explainable AI (feature importance + borrower-level drivers)
- FastAPI backend for real-time scoring
- Streamlit dashboard for user interaction

## Tech Stack
- Python
- Scikit-learn (Gradient Boosting)
- FastAPI
- Streamlit

## Use Case
Supports data-driven credit decision-making in SACCOs by predicting loan default risk.
